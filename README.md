# Student Management System (SQL Project)

## Description
This project is a database management system built using MySQL.
It manages students, courses, and enrollments.

## Features
- Create students and courses tables
- Insert and update data
- Use JOIN to combine tables
- Perform queries like filtering and grouping
- View course popularity and student enrollments

## Technologies Used
- MySQL
- SQL (Joins, Group By, Aggregate Functions)
